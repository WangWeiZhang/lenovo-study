一、宿主对象

解释：所谓宿主：就是一个生存环境。所以js的宿主绝大多数是浏览器。也就是由浏览器提供的对象。

  • Dom:document
  • Bomwindow

比如同样是document在chrome和firebug的功能是不一样的也就是随着运行环境不同会有所变化。

二、内置对象（静态对象）

解释：var oMath = new Math(); 我们平时没有这么写过，所以说他不需要实例化，可以直接使用。

  • Math：数学对象
  • Global：概念性词语，无法进行访问

总结：不需要实例化的对象就是 静态对象。

三、本地对象（非静态对象）

解释：如果某个语言还叫JavaScript的话，必须有以下对象。无论运行在什么上。规范里面规定的必须具备的对象。

  • Object
  • Function
  • Array
  • String
  • Number
  • Boolean
  • Date
  • RegExp
  • Error

总结：需要实例化的对象就是 非静态对象或者叫本地对象。
