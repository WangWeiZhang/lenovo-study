Css - Flex 使用总结

进入demo 利用v8查看 进入阮一峰 Css3 flex教程 
	父容器属性：
	
	flex:
		display:flex;
		display:-webkit-flex;
		
	flex-direction属性：定义排列方向
		row（默认值） 起点左面，从左到右，水平排列。
		row-reverse：起点右面，从右到左，水平排列。
		column：起点在上，从上到下，垂直排列。
		column-reverse：起点在下，从下到上，垂直排列。
	
	flex-wrap:如果一条轴线排不下，如何换行。
		nowrap：不换行。
		wrap：换行，第一行在上方。
		wrap-reverse：换行，第一行在下方。
	
	justify-content: 主轴对齐方式（正常水平）
		flex-start（默认值）：左对齐
		flex-end：右对齐
		center： 居中
		space-between：两端对齐，项目之间的间隔都相等。
		space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍
		
	align-items:交叉轴对齐方式
		flex-start：交叉轴的起点对齐。
		flex-end：交叉轴的终点对齐。
		center：交叉轴的中点对齐。
		baseline: 项目的第一行文字的基线对齐。
		stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度。
		
	align-content: 多条主轴，也就是换行，多行项目的时候生效
		flex-start：所有主轴整体 顶部对齐。（没边距）
		flex-end：所有主轴整体 底部对齐。（没边距）
		center：所有主轴整体 居中对齐；（没边距）
		space-between：所有主轴整体 两端对齐（两端没边距）
		space-around：所有主轴两端间距相等 。 所以，轴线之间的间隔比轴线与边框的间隔大一倍。
		stretch（默认值）：所有主轴，无论多少行，如果不设置高度或者是auto，那么会高度整体占满整个盒子。
		
	子容器的属性：
	
	order： 属性定义项目的排列顺序。数值越小，排列越靠前，默认为0。
		.item {
		   order: -1;   //如果其他项目不设置，则这个最小排在第一位置
		}
	
	flex-grow： 如何分配剩余空间。
		flex-grow:1;  //占有 父容器的 1份
		flex-grow:2;  //占有 父容器的 2份
		flex-grow:3;  //占有 父容器的 3份
	
	flex-shrink： 屏幕缩小时 是否等比缩小(需要结合 宽度的设置一起使用)
		flex-shrink：0;
		flex-shrink:1;
		flex-shrink:1;
		那么第一个等于0的将不会缩小，其余的会随着屏幕变化比例
	
	flex-basis属性： 再分配剩余空间的之前，占据主轴的空间。 可以使固定像素（类似于width和height 如果row就是width 如果 column 就是height）
		flex-basis:200px;
		flex-basis:400px;
	
	flex属性: flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。
		该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。
