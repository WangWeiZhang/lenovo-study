动态 CSS Class

示例 1

点击我改变颜色v-on:click="changeColorOne = {{changeColorOne}}"

总结：绑定class方法 v-bind:class="{changeColorOne:changeColorOne}" 双引号中是个对像

             key是class名称，value是布尔值，true添加类，false不添加

示例 2


点击上方按钮 添加颜色类和长度类

总结：既然v-bind:class="{object}" 我们可以将引号中写成方法，进行调用，在方法中使用return {将其返回}

             并且在触发 对象value的 true或false时，对原有的布尔值取反即可。
             这样可以一次添加多个动态类，并进行动态操作
